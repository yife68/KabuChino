# 西兰花

煮熟放凉，即可食用
吃的时候可以给他们掰碎一些

1~2朵 西兰花为宜

也可以放在猫粮里混一点

😺 卡布最爱的蔬菜~

# 白菜叶

使用盐清洗白菜表面，冲刷干净；必要的也可以泡一泡，泡久些
（为的是洗掉虫卵那些脏东西...

最后一定要清洗干净表面的盐份~

😼 奇诺最爱吃的蔬菜~ 所以俗称：白~菜喵~

# 生虾

可剥壳清理干净，切成小块 也可以稍微清理下，整只吃；看猫的喜好

奇诺不吃虾，笨笨的；卡布就把奇诺的也给吃了...

# 肉肉猫饭

注意：
猫饭有严格的比例，可以偶尔作为猫咪的加餐
一周 2~3次 即可，不建议长期吃~

猫饭的肉类占比为百分之八十左右；3~5种肉类最合适
由非鱼肉类 内脏类 海鱼类混合构成

可以是：
鸡 鸭 猪 兔 牛 羊 胸 心 胗 鸡蛋

重点提示：
各种 心脏 和 胗 都是肌肉组织，属于肉类
心脏牛磺酸含量超级丰富，是非常天然的牛磺酸补剂

但内脏的占比只能在 5-10% 之间，不可多出

我准备的食材：
鸡胸肉 鸡肝 鸡心 鸡胗 牛肉 蛋黄 胡萝卜 南瓜

使用搅拌机搅碎，或用到剁成小粒
放到蒸锅上蒸 15-25分钟，稍微放凉即可食用

# 鸡胸肉丝

菜市场买的一块鸡胸肉洗净 切成大拇指左右宽度

放入蒸锅蒸 15分钟 左右 取出放凉，撕成丝，倒上蒸出来的鸡汤，即可

补充营养最佳选择！！！

注：如猫咪不舒服或者在生病恢复期，可以将蒸出来的鸡汤喂它喝

# 牛奶鸡蛋饭

注意：
使用的一定要是纯牛奶！！！纯牛奶！！！

盆里倒入50毫升左右的 纯牛奶
将生鸡蛋的蛋黄倒入，进行搅拌
（注：是分离蛋清后的蛋黄！

可选择加入一点白米饭搅拌

# 鸡蛋鸡胸肉

我准备的食材：
两块鸡胸肉 大约500g
以及 一个鸡蛋

鸡胸肉洗净剁碎；将蛋清分离，只要蛋黄！！！
将 蛋黄 与 鸡胸肉碎 放碗里搅拌

可以吃的蔬菜也可以加一点进去
比如说胡萝卜、西兰花、卷心菜、南瓜等
不过在给猫咪喂食蔬菜时一定要煮熟煮透，猫咪不善于咀嚼是一方面，另一方面是猫咪的消化吸收能力也比较差

如果做多了，可以用小碗盛出当天的份量
其余的可以放到冰箱冷藏（冰冻冷藏）

放蒸锅蒸 15分钟左右 稍微放凉即可食用

# 蛋黄猫粮

鸡蛋黄 能提供丰富的维生素D，也是天然的化毛利器，一般一周可给2-3个蛋黄

可以直接给猫咪把蛋黄弄碎吃
也可以将蛋黄弄碎后，放入猫粮伴着吃
