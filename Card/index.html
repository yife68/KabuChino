# 🐾 起因

  • 21年7月9日

我在家楼下发现了两只流浪猫在嬉戏打闹  怪可爱的
但那只橘猫特别入我心，好几次都想带回家里去
碍于家里不便，却只能每日出门时碰上投喂些吃食

 

八月底，橘子却不见了
每每出门时，我都会留意留意附近是否有着流浪猫徘徊
直到一直晚上我回家时，在楼下遇到了这样一件事

我们家旁边是一个批发部，一个搬货的伙计
在仓库搬着货，猫咪在一边蹲着，静静着看着
当我刚走到家门前，突然听到一声惨烈的猫叫
谁曾想一转头就看到一扫把飞向那只猫咪
接踵，仓皇而逃

我不由衷的心里突然感到一惊
不知道橘子是不是也被这样欺负过，又或者说被这样赶跑了
毕竟在这附近怎么找都找不到它
况且橘子之前活动的地方就是在批发部门前不远的地方

  • 21.10.15 凌晨1点

直到一个多月后
这是一间大排档的杂货间，同时也是大排档的小灶间
再见到它时已是摇身一变成了五只幼猫的妈妈了

关于《小橘妈妈》 (opens new window)

环境的恶劣，加上生育，滋生细菌
使橘妈头上长满了猫藓，甚至还掉了许多的猫，触目惊心

而五只喵崽也有不同程度的猫藓
我知道，如果没有救助的话，这五只喵崽难以活过这寒冬


 
 

于是，17号晚上凌晨一点
我和女友一番讨论下，决定凌晨偷猫

没错，是"偷猫"
刚开始本来打算是偷一只
但中途想想还是决定再多偷一只，有个伴

就这样，我们收留了这两只幸运的小家伙
